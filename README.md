# multipleSite-Webscraping
Goal: Multi-Site Web Scraping &amp; Quantitative Claim Extraction

1) This project is a scalable web-scraping pipeline that automatically:

2) Discovers news and blog URLs using GDELT

3) Scrapes article content across multiple websites

4) Extracts quantitative productivity claims (percentages and time savings)

5) Captures contextual snippets explaining the claimed impact

6) Saves structured results to a CSV dataset
